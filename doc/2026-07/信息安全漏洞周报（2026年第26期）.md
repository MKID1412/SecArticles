1. Google Gemini CLI 输入验证错误漏洞（CNNVD-2026-40831485）  
  
Google Gemini CLI是美国Google公司开源的一款AI终端工具，支持在命令行中查询和编辑大型代码库、从图片或PDF生成应用，以及自动化复杂工作流。  
  
Google Gemini CLI 0.39.1之前版本存在输入验证错误漏洞，该漏洞源于对操作系统命令中和不当，攻击者利用该漏洞可以通过特制文件执行恶意代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://geminicli.com/  
  
2. Dell Wyse Management Suite 路径遍历漏洞（CNNVD-2026-10147733）  
  
Dell Wyse Management Suite是美国Dell公司的一套终端管理平台。  
  
Dell Wyse Management Suite 5.5 HF1之前版本存在路径遍历漏洞，该漏洞源于对路径名限制不当，攻击者利用该漏洞可以远程执行代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://www.dell.com/support/kbdoc/en-in/000465356/dsa-2026-225?msockid=3021cac2195069ed3194ddad186a68f9  
  
3. Linux kernel 安全漏洞（CNNVD-2026-26292867）  
  
Linux kernel是美国Linux基金会开源的一个Linux操作系统内核。  
  
Linux kernel 2.6.30版本存在安全漏洞，该漏洞源于nilfs_ioctl_mark_blocks_dirty函数未能正确验证bd_oblocknr参数为零的情况，攻击者利用该漏洞可以导致系统拒绝服务。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://www.kernel.org/

1. FlowiseAI Flowise 输入验证错误漏洞（CNNVD-2026-66011131）  
  
FlowiseAI Flowise是FlowiseAI公司开源的一个用于轻松构建LLM应用程序的工具。  
  
FlowiseAI Flowise 2.2.8版本及之前版本存在输入验证错误漏洞，该漏洞源于文件处理操作中缺少对chatflowId和chatId参数的验证，导致任意文件读写，攻击者利用该漏洞可以远程执行代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/FlowiseAI/Flowise/security/advisories/GHSA-q67q-549q-p849  
  
2. Microsoft Azure AI Bot Service 授权问题漏洞（CNNVD-2026-35403559）  
  
Microsoft Azure AI Bot Service是美国Microsoft公司的一套人工智能对话服务。  
  
Microsoft Azure AI Bot Service存在授权问题漏洞，该漏洞源于身份验证不当，攻击者利用该漏洞可以提升权限。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-32174  
  
3. OpenClaw 授权问题漏洞（CNNVD-2026-71993331）  
  
OpenClaw是一个开源的智能人工助理。  
  
OpenClaw 2026.5.7之前版本存在授权问题漏洞，该漏洞源于对账户身份验证不当，攻击者利用该漏洞可以提升权限。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/openclaw/openclaw/releases
