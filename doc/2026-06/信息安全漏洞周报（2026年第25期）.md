1. Google Android 授权问题漏洞（CNNVD-2026-39486590）  
  
Google Android是美国Google公司的一套以Linux为基础的操作系统。  
  
Google Android 17.0版本存在授权问题漏洞，该漏洞源于缺少权限检查，攻击者利用该漏洞可以泄露敏感信息。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://source.android.com/docs/security/bulletin/android-17  
  
2. Cisco Identity Services Engine Software 授权问题漏洞（CNNVD-2026-11229714）  
  
Cisco Identity Services Engine Software是美国Cisco公司的一套身份管理服务平台。  
  
Cisco Identity Services Engine Software存在授权问题漏洞，该漏洞源于访问资源时授权检查不当，攻击者利用该漏洞可以通过发送特制流量访问敏感信息，包括哈希凭据。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://sec.cloudapps.cisco.com/security/center/content/CiscoSecurityAdvisory/cisco-sa-ise-multi-G5WP8vv  
  
3. Microsoft Malware Protection Engine 后置链接漏洞（CNNVD-2026-49814918）  
  
Microsoft Malware Protection Engine是美国Microsoft公司的一套恶意软件防护检测引擎。  
  
Microsoft Malware Protection Engine存在安全漏洞，该漏洞源于文件访问前的链接解析不当，攻击者利用该漏洞可以提升权限。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-50656

1. Microsoft 365 Copilot 授权问题漏洞（CNNVD-2026-90090932）  
  
Microsoft 365 Copilot是美国Microsoft公司的一款AI生产力工具。  
  
Microsoft 365 Copilot存在授权问题漏洞，该漏洞源于关键功能缺少身份验证，攻击者利用该漏洞可以通过网获取露敏感信息。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-54130  
  
2. OpenClaw 权限许可和访问控制问题漏洞（CNNVD-2026-55449472）  
  
OpenClaw是一个开源的智能人工助理软件。  
  
OpenClaw 2026.5.2之前版本存在权限许可和访问控制问题漏洞，该漏洞源于搜索路径清理不当，攻击者利用该漏洞可以操控STATE_DIRECTORY变量，从而执行恶意代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/openclaw/openclaw/security/advisories/GHSA-wc84-j36w-pw4x  
  
3. FlowiseAIFlowise 跨站脚本漏洞（CNNVD-2026-50329025）  
  
FlowiseAI Flowise是FlowiseAI公司的一个用于轻松构建 LLM 应用程序的工具。  
  
FlowiseAI Flowise 3.0.8之前版本存在跨站脚本漏洞，该漏洞源于对聊天消息和自定义代理函数中的输入过滤不足，攻击者利用该漏洞可以注入恶意JavaScript脚本，从而窃取Cookie和会话数据。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/FlowiseAI/Flowise/security/advisories/GHSA-4fr9-3x69-36wv
