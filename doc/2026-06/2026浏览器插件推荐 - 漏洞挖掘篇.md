#### 4. Shodan  
  
Shodan插件是其网站服务的便捷入口，它的核心功能是让你能快速查询当前网站所在服务器的信息，点击插件图标，它会直接告诉你当前网页服务器的IP地址、开放端口、运行的服务和软件版本、可能存在的漏洞等信息，是网络安全研究人员进行信息收集的快速通道。

#### 5. Charset  
  
Charset是一个字符编码修复工具，它的核心功能是手动纠正网页的乱码问题，当浏览器因为自动识别错误而显示乱码时，你可以通过这个插件，从右键菜单里强制指定网页使用另一种字符编码（如UTF-8、GBK）来重新渲染，从而立刻解决乱码，让页面内容恢复正常显示。  
  
![](https://mmbiz.qpic.cn/mmbiz_png/tNyBeBKReNLT3qD7gvurVEIqFO5kCbbO4xLzjKQmUcPFJQibHOlNOwqEtYEb0mx3N5JQsgAt8Q7ibchJwcv1L66yjiadGicXf8J43gY0sHTicbZU/640?wx_fmt=png&from=appmsg "")  
  
#### 6. HackBar  
  
HackBar是一个面向渗透测试者的浏览器内置工具包，它的核心功能是提供一个独立的输入栏，用于手动构造和发送HTTP请求，特别是用于测试SQL注入、XSS等Web漏洞。它提供了快速加载特定攻击载荷、对URL和数据进行各种编码（Base64、URL编码等）和解码的功能，是手工安全测试的利器。  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tNyBeBKReNJ5IHQu3WVkM1RcjcI5m2icVCKAnyoHUDYqsBZwsEj964fvdYCGTnyL6ib3HR0Irx26Vicbic4TemelP0bibKZlVMCoriagP4efdWYGQ/640?wx_fmt=png&from=appmsg "")

WebSocket DevTools  
  
WebSocket DevTools是一个专业WebSocket调试插件，具备代理、模拟、阻断消息，高级调试等功能，兼容所有WebSocket实现，包括原生WebSocket API、Socket.IO和ws库，让WebSocket调试在现代Web开发工作流程中变得简单高效。  
  
![](https://mmbiz.qpic.cn/mmbiz_png/tNyBeBKReNJsqSkdHmicLUXIucklibDMjsONJ8GxsKlhuFdw6AFFZ8czW97cicbnicFFx7PJFloZrZicpJ7e71NSicb50fibDpfdugTm9WUwAw30icQ/640?wx_fmt=png&from=appmsg "")  
  
#### 26. Ctool 程序开发常用工具  
  
Ctool是一个面向程序员的常用工具集合，它的核心功能是将开发中频繁用到的小功能整合在一起，例如JSON格式化与校验、时间戳转换、Base64/URL编解码、正则表达式测试、二维码生成、哈希计算等，免去了你反复搜索在线工具网站的麻烦。  
  
![](https://mmbiz.qpic.cn/mmbiz_png/tNyBeBKReNLI85Yx11DepsCO3f3UImsgy9arDJuLoPzIQ4D4QfnHqH7NyWEFpqYde4417u55fXoB6FjhyLUJstqTX2A5nXibem0Y51IfTcOE/640?wx_fmt=png&from=appmsg "")  
####   
#### 27. Wappalyzer - Technology profiler  
  
Wappalyzer是一个网站技术栈识别工具，它的核心功能是分析当前网页，识别并告诉你该网站是用什么技术搭建的，包括内容管理系统（CMS）、Web框架、JavaScript库、分析工具、支付网关等，并在插件图标上直接显示主要技术的logo，是技术调研和竞品分析的利器。  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tNyBeBKReNJsyQNReZYyQxjqrpy7Kib9o3BocAxBp1ABwZMjq52kEg8EkP4WjqZvxLohKO1alFhf9iaSicz4kPiaZaneURLFYYUcwa5qYOXDBuM/640?wx_fmt=png&from=appmsg "")  
####   
#### 28. Weak Password Dict Generator  
  
无境剑（Weak Password Dict Generator）是一个企业弱口令字典生成浏览器插件，可以基于域名和自定义关键字自动生成企业弱口令字典，用于授权的渗透测试和安全评估。  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tNyBeBKReNIbcOIA4865AnW6Eyz3EChPibq9m39V39NzwFcfoWZ5ibibxBbNWQ5xQypmtaON7VXJ1Oiaiaic5o4Sr8jYWJklEtlcKdzZx7ibyKImuY/640?wx_fmt=png&from=appmsg "")  
####   
#### 29. WPBurp - Weak Password Checker  
  
WPBurp是一个弱密码检测工具，它的核心功能是在浏览器端自动化地检测登录表单是否使用了常见弱口令，或者基于自定义字典对网站后台、管理面板进行简单的弱口令爆破测试。  
  
![](https://mmbiz.qpic.cn/mmbiz_png/tNyBeBKReNITL3uQ9tw0g1ZXC8vD9rfZyhnyG3vEcDvDoDZUDA8J0ylY0jr8A3xBWSUT76AfD8biaUKic31bzRtBHWtATZBgy1qnFGBurNOgk/640?wx_fmt=png&from=appmsg "")  
####   
#### 30. User-Agent Switcher and Manager  
  
User-Agent Switcher and Manager是一个浏览器身份伪装工具，它的核心功能是让你能够修改浏览器发送给网站的User-Agent字符串，你可以从预设的浏览器和蜘蛛UA列表中选择，也可以自定义UA字符串，并能为特定的网站设置独立的UA规则，常用于测试移动端网页或绕过某些限制。  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tNyBeBKReNLqG26ACGzNonFDzSWCwu4UwUXh9J1w3Ujv14fFF6Tv773mT3Z7fwGHc0DxkhEfAUDkTicuApDx7tEWYs3ezNeVlYNDLd6vT0mg/640?wx_fmt=png&from=appmsg "")  
####   
#### 31. ModHeader - Modify HTTP headers  
  
ModHeader是一个HTTP请求/响应头修改工具，它的核心功能是让你能够添加、修改或删除浏览器发出的请求头和接收到的响应头，并且修改是即时生效、无需刷新的。这对于前后端联调、测试特定头部的CORS、安全策略或绕过IP限制等开发场景非常实用。  
  
![](https://mmbiz.qpic.cn/mmbiz_png/tNyBeBKReNJfeB6icP6FQajU2duIQThmum91dicZKumcGa04OfiaUUS8tuUAQvlkmJAHJ0gPnvv0Z18sPicdbq7g88ZMK6jJSeIPzKT9oDFFBv8/640?wx_fmt=png&from=appmsg "")  
####   
#### 32. Proxy SwitchyOmega 3 (ZeroOmega)  
  
ZeroOmega是SwitchyOmega的一个现代维护分支，它的核心功能是轻松管理和切换代理服务器，你可以创建多个代理配置文件（如直连、系统代理、不同的代理服务器），并根据访问的网站域名自动应用不同的代理规则，界面清晰，逻辑强大，是代理管理的标杆工具。  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/tNyBeBKReNLyXQVjffXtTicuhVgDZYtbruPHXb20jMzvTdw4icpxcZfo5hXPzjIO7n5W1zibGiaDXVDhRbAFJex3VBd42icWRiazCHzHA08ATc6J4/640?wx_fmt=png&from=appmsg "")
