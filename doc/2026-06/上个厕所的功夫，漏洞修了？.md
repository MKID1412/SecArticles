Chrome 有 Cache 缓存机制，想在缓存中恢复网页还是有可能的。  
  
先确定思路  
  
```
Windows 的 Chrome 的 cache 路径
C:\Users\用户名\AppData\Local\Google\Chrome\User Data\Default\Cache
```  
  
```
MacOS 的 Chrome 的 cache 路径
~/Library/Caches/Google/Chrome/Default/Cache
```  
  
cache data 数据如下  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/a3etiafIAYXHakqTicf2qUWWlcbvZLuk3AoXZr0pibz4d5iabGicXpKibyVmy4SrcNEbPDQlocOdX0mSTQtUPqOJ7d9CF1SomF8cr4YYBAYul4VFE/640?wx_fmt=png&from=appmsg "")  
  
Chrome 缓存内部结构：  
  
data_0  —— 条目索引（存 URL 和"内容在哪儿"的指针）  
  
data_1  —— 256B 块（HTTP 响应头 + 小文件）  
  
data_2  —— 1024B 块（中等文件）  
  
data_3  —— 4096B 块（大文件：HTML 页面、大图片等）
