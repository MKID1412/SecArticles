```
sql =
"SELECT ... WHERE dtname="+dtname +"where ID>0 order by OrderBy"
;
```  
  
查看数据库为sql server2012  
```
(select+1+ID,0+SID,@@version+CName,1+OrderBy)a
```  
  
  
![](https://mmbiz.qpic.cn/mmbiz_png/kaTLeqcBezicJrCibeYflT8icwbbJNicaOG4tjzSiaAbHrDk9rScMW5Jnm7dHcWfwnEca2ENWsjKOBfib1aH69nVclImEMRg2kuM2w4ACIiciaJbftE/640?wx_fmt=png&from=appmsg "")
