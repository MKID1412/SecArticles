#  关注丨通过备案的网络产品安全漏洞收集平台（第1批）  
 工业安全产业联盟平台   2026-05-20 10:49  
  
根据《中华人民共和国网络安全法》《工业和信息化部 国家互联网信息办公室 公安部关于印发网络产品安全漏洞管理规定的通知》（工信部联网安〔2021〕66号）《工业和信息化部关于印发〈网络产品安全漏洞收集平台备案管理办法〉的通知》（工信部网安〔2022〕146号），工业和信息化部现将通过备案的网络产品安全漏洞收集平台（第1批）予以公布。
